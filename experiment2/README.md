

##  Aim

To design and develop a **modern, responsive, and visually appealing web page** using a **UI component library** in React.

---

##  Objective

* To understand **component-based UI development**
* To learn **folder structuring in React**
* To design a **real-world meaningful webpage**
* To apply **modern UI/UX principles**
* To use a **UI component library (Material UI)**

---

##  Technologies Used

* React (Vite)
* Material UI (MUI)
* JavaScript (ES6)
* HTML & CSS

---

##  UI Library Used

* **Material UI (MUI)**

---

##  Folder Structure

```
src/
│
├── Components/
│   ├── Navbar.jsx
│   ├── HeroSection.jsx
│   ├── CardComponent.jsx
│   └── Footer.jsx
│
├── pages/
│   └── Home.jsx
│
├── App.jsx
├── main.jsx
└── index.css
```

---

##  Description of Components

* **Navbar** – Displays navigation bar with site title
* **HeroSection** – Main banner section with heading and call-to-action
* **CardComponent** – Reusable card for displaying features
* **Footer** – Footer section with copyright text
* **Home Page** – Combines all components into a complete webpage

---

## ⚙️ Installation Steps

1. Create React project using Vite

   ```
   npm create vite@latest
   ```
2. Navigate to project folder

   ```
   cd experiment2
   ```
3. Install dependencies

   ```
   npm install
   ```
4. Install Material UI

   ```
   npm install @mui/material @emotion/react @emotion/styled
   ```
5. Run the project

   ```
   npm run dev
   ```
